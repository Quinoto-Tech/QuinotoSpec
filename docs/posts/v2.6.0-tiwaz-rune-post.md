## Las 3 fases de Tiwaz

### 🔥 1. Entropia de Shannon (conocete a ti mismo)

Tiwaz construye un grafo de tu codigo — cada modulo un nodo, cada import una arista — y aplica `H(G) = -Σ (grado(n) / 2|E|) * log2(grado(n) / 2|E|)`. Mide entropia ciclomatica (`log2(n)` por funcion), distribucion de tamanos (Huberman-Hargitai), churn por git log, y fragmentacion de configuracion entre entornos. Cosas que SonarQube no ve.

### ⚔️ 2. Deuda tecnica (el combate)

7 dimensiones medibles en minutos: Complejidad, Acoplamiento, Configuracion, Dependencias, Seguridad, Testing, Observabilidad. No es teoria — son archivos, lineas, secretos expuestos.
